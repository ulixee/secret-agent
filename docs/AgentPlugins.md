# Agent Plugins

Agent plugins call installed "hooks" at the appropriate moments in the initialization of a [Browser](./Browser.md), [BrowserContext](./BrowserContext.md), [Page](./Page.md), [Worker](./Worker.md) or Network request.

Hooks can be used to emulate a Browser Engine or a Human user.

Unblocked Agent Plugins implement the Specification defined [here](https://github.com/unblocked-web/specifications).
